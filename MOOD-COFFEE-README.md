## 项目简介

这是 MOOD COFFEE（小红店）的官方主页网站，采用单页滚动式设计，暗黑霓虹风格，匹配店铺黑底红字招牌的视觉调性。

**线上预览**：直接用浏览器打开 `mood-coffee.html` 即可查看。

| 元素 | 说明 |
|------|------|
| **主色调** | 黑底 `#0a0a0a` + 红橙霓虹渐变 `#ff0040 → #ff6600` |

## 技术实现

- **纯 HTML/CSS/JS**，单文件，零依赖
- 响应式设计，适配桌面端和移动端
- 移动端汉堡菜单
- CSS 变量管理主题色，方便换肤
- Intersection Observer 实现滚动渐入动画
- 导航栏滚动变色（背景模糊 + 边框）

### 文件结构
```
mood-coffee.html    ← 单文件，包含全部 HTML + CSS + JS
```

## 生成过程记录

### 数据来源
1. **店铺门面照片**：`MOOD COFFEE.jpg`（桌面）—— 用于 Hero 背景和关于区展示
2. **咖啡菜单**：`咖啡.xmind`（桌面）—— 提取 content.json 获取饮品名称和分类
3. **鸡尾酒菜单**：`鸡尾酒.xmind`（桌面）—— 提取无酒精/微醺/烈酒/特调四大类
4. **现有酒单**：`现有酒单.xmind`（桌面）—— 提取精选招牌款

### xmind 数据提取方式
```bash
# xmind 本质是 zip 包，解压后读取 content.json
unzip -o "鸡尾酒.xmind" -d /tmp/xmind_extract/cocktail/
cat /tmp/xmind_extract/cocktail/content.json
```
